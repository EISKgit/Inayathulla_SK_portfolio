# React Portfolio (Vite + Tailwind)

Generated portfolio scaffold (modular) with Tailwind dark-mode (class strategy).

## Setup
1. `cd react-portfolio`
2. `npm install`
3. `npm run dev`

## Notes
- Theme toggling uses Tailwind's `dark` class and stores preference in `localStorage`.
- Content is separated in `src/data/*`. Edit those files to update resume content without touching UI.
